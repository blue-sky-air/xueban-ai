## 📋 步骤2：DNS解析配置（需你操作）

**要求**: 保持服务解析至腾讯云服务器，且备案信息真实有效

**操作步骤**:

1. **登录腾讯云控制台**
   - 访问: https://console.cloud.tencent.com/cns

2. **添加域名解析记录**
   ```
   主机记录: @      记录类型: A    记录值: 你的服务器公网IP
   主机记录: www    记录类型: A    记录值: 你的服务器公网IP
   ```

3. **验证解析是否生效**
   ```bash
   # 在本地电脑执行
   nslookup bblue.art
   ping bblue.art
   ```

4. **申请HTTPS证书**（推荐）
   ```bash
   # 在服务器上执行
   sudo certbot --nginx -d bblue.art -d www.bblue.art
   ```

**注意事项**:
- 备案信息必须真实有效，如有修改需进行变更备案操作
- 域名解析必须指向腾讯云服务器（备案所在服务器）

## 📋 步骤3：公安联网备案（需你操作）

**要求**: 需在服务开通三十日内进行公安联网备案

**操作步骤**:

1. **访问全国互联网安全管理服务平台**
   - 网址: https://www.beian.gov.cn/

2. **注册/登录账号**
   - 使用企业或个人信息注册

3. **提交备案申请**
   - 填写网站信息（域名、IP地址等）
   - 填写主体信息（个人/企业信息）
   - 上传相关证件

4. **等待审核**
   - 一般需要 5-20 个工作日

**所需材料**（个人备案）:
- 身份证正反面照片
- 域名证书
- 网站负责人照片（部分地区需要幕布照）

**所需材料**（企业备案）:
- 营业执照副本照片
- 法人身份证正反面照片
- 网站负责人身份证正反面照片
- 域名证书
- 网站负责人照片

**处罚说明**:
- 如不履行，会给予警告
- 严重的停机整顿不超过六个月

## 📞 常见问题

**Q: 备案号在哪里显示？**
A: 在网站每个页面的底部，点击可跳转到工信部备案官网。

**Q: 域名解析不生效怎么办？**
A: DNS解析需要时间生效，一般几分钟到24小时。可以清除本地DNS缓存后重试。

**Q: 公安备案和ICP备案有什么区别？**
A: ICP备案是工信部要求，公安备案是公安部要求，两者都需要完成。

**Q: 备案信息变更了怎么办？**
A: 需要及时在原备案系统提交变更申请，否则可能被注销备案。
